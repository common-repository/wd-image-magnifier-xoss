== Description ==


Wd-image-magnifier-xoss is a very simple image magnifier wordpress plugin with hover.This plugin help you to add a image magnifier in your wordpress website.

plugin Demo : http://wordpressdriver.com/test/wd-magnifier-test/

== Installation ==


Wd-image-magnifier-xoss installation guideline:

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
4.this plugin you can use in your page,post,widget. 

shortcode:[magni id="id name" link="your image link"]

example;[magni id="zoomwd1" link="http://wordpressdriver.com/blue.jpg"]
        [magni id="zoomwd2" link="http://wordpressdriver.com/read.jpg"]
		
if you want to add your magnifier width and height just define width height value.

example: [magni width:"150" height="100" id="anotherid" link="http://wordpressdriver.com/blue.jpg"]

If you want to use shortcode in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` 
in your templates.

And just hover on the image.
